# Nexus Chat

Чат-клиент и сервер на C# (.NET 10) с TCP-сокетами.



============================================
**Сайт:** http://51.250.47.167:8080
====================================



## Возможности
- Регистрация / вход
- Комнаты (create / join / leave)
- Личные сообщения (/w ник текст)
- Отложенные сообщения (/delay сек текст)
- Отправка изображений (/upload, /i, /view)
- Офлайн-сообщения
- Shh-режим (/mute)

## Запуск сервера
```bash
./ChatServer [порт]  # по умолч. 8888
```

## Подключение клиента
```
IP: 51.250.47.167
```

## Сборка
```bash
dotnet build ChatServer/ChatServer.csproj
dotnet build ChatClient/ChatClient.csproj
```
