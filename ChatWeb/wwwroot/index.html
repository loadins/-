Nexus Chat — как пользоваться

  • Сообщения — просто пиши и жми Enter
  • Файлы — выбери файл, нажми "Загрузить"
    ссылка появится в чате
  • Комнаты — кликни по комнате слева чтобы переключиться

Команды

  • /help — показать эту справку
  • /clear — очистить чат (локально)
  • /users — список участников в комнате
  • /files — список твоих файлов

Кнопки

  • Шёпот — личное сообщение выбранному юзеру
  • Ответ — ответить в комнату (выделенный ответ)
  • Создать — создать новую комнату
  • Комнаты — список всех комнат
  • Ливнуть — выйти в General
  • Файлы — список твоих файлов
  • Задержка — отправить сообщение через N секунд
  • Переслать — отправить свой файл другому юзеру
